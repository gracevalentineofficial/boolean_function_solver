Hukum Aljabar Boolean

Tabel Hukum Aljabar Boolean:

No Nama Hukum OR AND
1 Identitas x + 0 = x x · 1 = x
2 Idempoten x + x = x x · x = x
3 Komplemen x + x' = 1 x · x' = 0
4 Dominansi x + 1 = 1 x · 0 = 0
5 Involusi (x')' = x
6 Penyerapan x + (x · y) = x x · (x + y) = x
7 Komutatif x + y = y + x x · y = y · x
8 Asosiatif x + (y + z) = (x + y) + z x · (y · z) = (x · y) · z
9 Distributif x + (y · z) = (x + y) · (x + z) x · (y + z) = (x · y) + (x · z)
10 De Morgan (x + y)' = x' · y' (x · y)' = x' + y'
11 0 / 1 1' = 0 0' = 1

Fungsi Boolean

Dinyatakan dalam bentuk: f(variabel) = ekspresi
Ekspresi terdiri atas:
  • Operator: Menggunakan simbol + (OR), * / · (AND), serta ' (NOT).
  • Variabel: Huruf alfabet tunggal yang bernilai digital mutlak 0 atau 1.
  • Konstanta: Ketetapan nilai logika biner berupa angka 0 atau 1.
  • Pengelompokan: Mengatur prioritas operasi menggunakan tanda kurung ().

Referensi Operator:

Operasi Simbol Deskripsi Contoh
AND &, ·, *, AND, && Konjungsi logis A & B, A * B
OR +, |, v, OR, || Disjungsi logis A | B, A + B
NOT !, ~, ', ¬, NOT Negasi logis !A, A'
XOR ⊕, ^, XOR Eksklusif OR A ⊕ B, A ^ B
NAND ↑, NAND NOT AND A ↑ B
NOR ↓, NOR NOT OR A ↓ B
