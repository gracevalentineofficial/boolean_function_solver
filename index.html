Hukum Aljabar Boolean

Berikut adalah 11 hukum formal dalam Aljabar Boolean:

No Nama Hukum Ekspresi
1Identitasx + 0 = x, x · 1 = x
2Null (Dominasi)x + 1 = 1, x · 0 = 0
3Idempotenx + x = x, x · x = x
4Komplemenx + x' = 1, x · x' = 0
5Involusi(x')' = x
6Penyerapanx + (x · y) = x, x · (x + y) = x
7Komutatifx + y = y + x, x · y = y · x
8Asosiatifx + (y + z) = (x + y) + z
9Distributifx + (y · z) = (x + y) · (x + z)
10De Morgan(x + y)' = x' · y', (x · y)' = x' + y'
11Konsensus(x · y) + (x' · z) + (y · z) = (x · y) + (x' · z)
