Hukum Aljabar Boolean

Aljabar Boolean memiliki 11 hukum formal sebagai berikut:

1. Hukum Identitas: x + 0 = x, x · 1 = x
2. Hukum Null (Dominasi): x + 1 = 1, x · 0 = 0
3. Hukum Idempoten: x + x = x, x · x = x
4. Hukum Komplemen: x + x' = 1, x · x' = 0
5. Hukum Involusi (Negasi Ganda): (x')' = x
6. Hukum Penyerapan (Absorpsi): x + (x · y) = x, x · (x + y) = x
7. Hukum Komutatif: x + y = y + x, x · y = y · x
8. Hukum Asosiatif: x + (y + z) = (x + y) + z, x · (y · z) = (x · y) · z
9. Hukum Distributif: x + (y · z) = (x + y) · (x + z), x · (y + z) = (x · y) + (x · z)
10. Hukum De Morgan: (x + y)' = x' · y', (x · y)' = x' + y'
11. Hukum Konsensus: (x · y) + (x' · z) + (y · z) = (x · y) + (x' · z)
