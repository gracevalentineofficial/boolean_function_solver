Nama Hukum Dua-an (Duality 1) Dua-an (Duality 2)
Identityx + 0 = xx * 1 = x
Idempotentx + x = xx * x = x
Complementx + x' = 1x * x' = 0
Involution (Double NOT)(x')' = x
Commutativex + y = y + xx * y = y * x
Associativex + (y + z) = (x + y) + zx * (y * z) = (x * y) * z
Distributivex + (y * z) = (x + y)*(x + z)x * (y + z) = (x * y) + (x * z)
De Morgan(x + y)' = x' * y'(x * y)' = x' + y'
Absorption (Penyerapan)x + (x * y) = xx * (x + y) = x
