📜 1. Hukum-Hukum Aljabar Boolean

Aljabar Boolean memiliki properti dasar yang diatur oleh 11 hukum formal (Prinsip Dualitas). Hukum ini digunakan untuk memanipulasi ekspresi logika tanpa mengubah nilai kebenarannya.

border="1"
Nama HukumDualitas 1Dualitas 2
1. Hukum Identitasx + 0 = xx · 1 = x
2. Hukum Idempotenx + x = xx · x = x
3. Hukum Komplemenx + x' = 1x · x' = 0
4. Hukum Involusi(x')' = x
5. Hukum Komutatifx + y = y + xx · y = y · x
6. Hukum Asosiatifx + (y + z) = (x + y) + zx · (y · z) = (x · y) · z
7. Hukum Distributifx + (y · z) = (x + y)(x + z)x · (y + z) = (x · y) + (x · z)
8. Hukum De Morgan(x + y)' = x' · y'(x · y)' = x' + y'
9. Hukum Penyerapan (Absorpsi)x + (x · y) = xx · (x + y) = x
10. Hukum Dominansi (0 dan 1)x + 1 = 1x · 0 = 0
11. Hukum Komplemen Ganda 0/10' = 11' = 0

📝 2. Fungsi Boolean & Substitusi Nilai

Fungsi Boolean adalah ekspresi yang dibentuk oleh literal (variabel) dan operator logika (+, ·, '). Nilai evaluasi fungsi selalu berupa elemen biner biner {0, 1} tergantung nilai substitusi komponennya.

🔌 4. Skematik Rangkaian Gerbang Logika

Fungsi Boolean dapat diimplementasikan ke dalam perangkat keras digital melalui gerbang logika standar: AND (perkalian), OR (penjumlahan), dan NOT (pembalik sinyal).
