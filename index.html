Hukum Aljabar Boolean

Aljabar Boolean memiliki properti dasar yang diatur oleh 11 hukum formal (Prinsip Dualitas). Hukum ini digunakan untuk memanipulasi ekspresi logika tanpa mengubah nilai kebenarannya.

Nama HukumDualitas 1Dualitas 2
Hukum Identitasx + 0 = xx · 1 = x
Hukum Idempotenx + x = xx · x = x
Hukum Komplemenx + x' = 1x · x' = 0
Hukum Involusi(x')' = x
Hukum Komutatifx + y = y + xx · y = y · x
Hukum Asosiatifx + (y + z) = (x + y) + zx · (y · z) = (x · y) · z
Hukum Distributifx + (y · z) = (x + y)(x + z)x · (y + z) = (x · y) + (x · z)
Hukum De Morgan(x + y)' = x' · y'(x · y)' = x' + y'
Hukum Penyerapan (Absorpsi)x + (x · y) = xx · (x + y) = x
Hukum Dominansix + 1 = 1x · 0 = 0
Hukum Komplemen Ganda0' = 11' = 0

Fungsi Boolean

Fungsi Boolean adalah ekspresi yang dibentuk oleh literal (variabel) dan operator logika (+, ·, '). Nilai evaluasi fungsi selalu berupa elemen biner {0, 1} tergantung nilai substitusi komponennya.
